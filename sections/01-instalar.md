## Instalar Rust 

~~~shell
Requisitos previos de Rust -> Visual C++

Rust requiere un enlazador y bibliotecas API de Windows, pero no parecen serlo.
disponible.

Estos componentes se pueden adquirir a través de un instalador de Visual Studio.

1) Instalación rápida a través del instalador de Visual Studio Community
   (gratis para particulares, usos académicos y código abierto).

2) Instalar manualmente los requisitos previos
   (para usuarios empresariales y avanzados).

3) No instale los requisitos previos
   (si está apuntando a GNU ABI).
   
~~~
